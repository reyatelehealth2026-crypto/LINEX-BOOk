Elevator Pitch (30 วินาที)

"ลูกค้าจองคิวผ่าน LINE ได้ใน 30 วินาที ไม่ต้องโหลดแอป
เจ้าของร้านดูคิว Real-time ผ่านมือถือ — จ่ายครั้งเดียว ไม่มีรายเดือน"
Customer books in 30 sec via LINE · Owner sees live queue · One-time fee, no monthly lock-in
💡
Key insight: คู่แข่งทุกเจ้า (FoxConnect, QueQ, Agency จ้างทำ) ต้องให้ลูกค้า "ออกจาก LINE" เพื่อจอง — LineBook ไม่ต้อง

5 จุดเด่นที่คู่แข่งทำไม่ได้ (Technical Moat)

💬
LINE-Native: ไม่ต้องออกจากแอป
ลูกค้าจองผ่าน LIFF ภายใน LINE chat ได้เลย ไม่ต้องโหลดแอปเพิ่ม ไม่ต้องจำ URL
❌ คู่แข่ง: ต้องส่ง link ออกไปเว็บ/แอปอื่น
✅ LineBook: อยู่ใน LINE ตลอด
🧠
พิมพ์ภาษาไทยธรรมชาติได้
"พรุ่งนี้บ่ายสองตัดผมกับน้องแนน" → ระบบเข้าใจได้เลย ไม่ต้องกดปุ่มทีละขั้น
❌ คู่แข่ง: กดเลือกจาก form หรือ menu เท่านั้น
✅ LineBook: NLP parser ภาษาไทย
🛡️
DB-Level Double-Booking Prevention
PostgreSQL exclusion constraint หมายความว่าจองทับกันไม่ได้ แม้มีคน 100 คนกดพร้อมกัน
❌ คู่แข่ง: เช็ค application layer อาจ race condition
✅ LineBook: database จัดการ
⚙️
ตั้งค่าร้านผ่าน LINE Chat
เจ้าของร้านตั้งค่าบริการ ราคา ช่าง และเวลาทำการได้ผ่านแชทโดยไม่ต้องเข้าเว็บ
❌ คู่แข่ง: ต้องเข้า web dashboard เสมอ
✅ LineBook: LINE chat wizard
💰
ราคา One-Time Fee ไม่มีรายเดือน
จ่ายครั้งเดียว เจ้าของร้านเป็นเจ้าของระบบและข้อมูลลูกค้าทั้งหมด
❌ FoxConnect: ฿500–2,000/เดือน ตลอดชีพ
✅ LineBook: จ่ายครั้งเดียว ฿4,900–49,900

Impact × Effort Matrix — ลำดับการพัฒนาแนะนำ

🚀 ทำก่อน — Impact สูง / Effort ต่ำ
  • Cron reminders API route (~2 ชม.)
  • Reminder 24 ชั่วโมง (~3 ชม.)
  • Demo seed script (~2 ชม.)
  • Review request push (~3 ชม.)
  • Revenue card บน dashboard (~4 ชม.)
📅 วางแผน — Impact สูง / Effort สูง
  • Waitlist auto-notification
  • Refactor webhook เป็น modules
  • Multi-stage reminders (24h / 2h / 30m)
  • Customer profile timeline
  • Tier system (Silver/Gold/VIP)
⚡ Fill-in — Impact ต่ำ / Effort ต่ำ
  • Error message ภาษาไทยทุกจุด
  • Loading skeleton บน LIFF
  • Admin healthcheck: cron last-run
  • vercel.json สำหรับ cron
🔮 Long-term — Impact ต่ำ / Effort สูง
  • Voice-to-booking (Whisper API)
  • Multi-shop mode
  • LINE Pay / PromptPay deposit
  • Before/After photo gallery
  • Automated unit test suite
